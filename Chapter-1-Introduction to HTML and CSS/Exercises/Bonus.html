HTML Tags

HEAD

Contains metadata about the HTML document.

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML & CSS Quick-Reference Guide</title>
    <link rel="stylesheet" href="Bonus.css">
</head>

TITLE

Sets the title of the HTML document.

<title>HTML & CSS Quick-Reference Guide</title>

BODY

Contains the content of the HTML document.

<body>
    ...
</body>

H1 TO H6

Define headings of different levels, with <h1> being the most important and <h6> the least.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
...
<h6>Heading 6</h6>

P

Defines a paragraph.

<p>This is a paragraph.</p>

A

Creates a hyperlink to another web page or resource.

<a href="https://example.com">Link Text</a>

IMG

Embeds an image in the document.

<img src="image.jpg" alt="Image Description">

UL

Defines an unordered list.

<ul>
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>

OL

Defines an ordered list.

<ol>
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ol>

LI

Defines a list item within <ul> or <ol>.

<ul>
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>

DIV

Defines a division or section in the document.

<div>
    ...
</div>

SPAN

Defines a section in the document for styling purposes.

<span>This is a styled section.</span>

TABLE

Defines a table.

<table>
        <thead>
            <tr>
                <th>Header 1</th>
                <th>Header 2</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Data 1</td>
                <td>Data 2</td>
            </tr>
        </tbody>
    </table>

CSS Properties

Color

Sets the text color.

Font-Family

Defines the font family for text.

Background-Color

Sets the background color of an element.

Padding

Sets the padding space inside an element.

Margin

Sets the margin space outside an element.

Border

Specifies the border of an element.

Width

Sets the width of an element.

Height

Sets the height of an element.

Display

Specifies how an element is displayed.

Position

Defines the positioning method used for an element.
